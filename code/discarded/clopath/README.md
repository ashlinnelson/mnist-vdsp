This folder contains an attempt to build an unsupervised Spiking Neural Network (SNN) for MNIST digit classification using the Voltage-Dependent Synaptic Plasticity (VDSP) rule proposed by Clopath et al. (2010).

The network fundamentally fails to specialize due to a mismatch between the dataset's sparsity and the plasticity rule's mechanics. In the Clopath model, Long-Term Depression (LTD) is strictly triggered by presynaptic spikes, specifically when a presynaptic signal arrives shortly after the postsynaptic neuron is depolarized. However, MNIST background pixels are completely black, meaning their Poisson firing rate is 0 Hz. Because these silent pixels never generate spikes, they never trigger the necessary on_pre events. As a result, LTD is never applied to the negative space.

Without LTD to carve out inactive regions, center pixels continually accumulate Long-Term Potentiation (LTP) from various digits without ever losing weight due to LTD. The synaptic weights permanently max out, causing all neurons to develop nearly identical, generic receptive fields. Because the neurons fail to differentiate, the Winner-Take-All (WTA) lateral inhibition mechanism completely breaks down: every neuron attempts to fire at the exact same dense central pixels, resulting in total network failure.
